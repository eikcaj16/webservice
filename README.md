## Prerequisites

- Linux, macOS, Windows
- Python 3.6-3.8



## Usage

####1. Install all the necessary packages

```bash
pip install django djangorestframework
```

####2. Deploy the web application

```bash
python manage.py runserver
```

#### 3. Run Unit Test (Optional)

```bash
python manage.py test
```
